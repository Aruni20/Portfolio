# Aruni Saxena — Portfolio

Personal portfolio and project showcase. Built with **React 19**, **Vite 7**, **Framer Motion**, and **Lucide icons**.

🔗 **Live:** [aruni-quant-portfolio.surge.sh](https://aruni-quant-portfolio.surge.sh)

---

### Run locally

```bash
npm install
npm run dev
```

### Build for production

```bash
npm run build
```

### Deploy (Surge)

```bash
npm run build
npx surge ./dist your-domain.surge.sh
```

---

### Structure

- **Hero & intro** — Markets, models, machine intelligence; quant / ML research / applied AI pillars.
- **Experience** — SMC Global, QuantX Technology, Smart Energy Learning Centre (with bullet summaries).
- **Contributions** — Filterable project grid (Quant Finance, Generative AI, Data Science, Engineering) with problem/approach details and GitHub links.
- **Education** — M.Sc. Data Science (DAIICT), B.Sc. (Hons) Mathematics (JMC, DU).
- **Honors** — Academic work, leadership, service, creative.
- **Skills** — Quantitative Finance, Generative AI, Data Science, Systems & Infrastructure.
- **Connection** — Reach-out form (Formspree) and footer links.

Data is driven by `src/data/` (experience, education, projects). Replace `YOUR_FORM_ID` in the contact form with your [Formspree](https://formspree.io) form ID to receive messages.
